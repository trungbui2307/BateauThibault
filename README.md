## Téchnology
Projet est créé avec:
#### Framework:
    * Django
    * Angular
#### Langages:
    * Python
    * Typescipt
    * Html/css
    
## Installer
Pour exécuter ce projet, installez-le localement à l'aide de npm :
```
$ npm install - g @angular/cli
$ cd FrontendAngular
$ ng serve
```
